Explore by Category

Dive deeper into specific functionality areas

Offline IoT Management

Includes a web app that can be installed on each device to allow IoT management. Useful for network constained environments with few IoT devices. Part of our free-tier plan.

System Statistics

Check service logs, running services, CPU, memory, and storage usage. Allows time synchronization and device reboot.

Network Management

Configure static IP or DHCP on the connected network interfaces. Add local DNS entries and check for outbound internet connection.

Updater Utility

Upload a bundle that allows you to install custom software on the IoT device (e.g. programs to read sensor data).

User Management

Roll based access control to access the web portal. Useful to allow clients to perform basic system maintenance.

File Explorer

Allows users to transfer files to and from the device. Allows technicians to easily offload data from the IoT device.

Web Terminal

If advanced interaction is needed, we provide a web terminal to the machine to run commands on the operating system.

Self-Hosted (Coming Soon)

Software that can be installed on a device to manage a large fleet of IoT devices. Useful to manage a suite of devices on a restricted network. Part of our paid plan.

Firewall Friendly

If being installed on a client network, the IoT devices can connect to this server without worrying about outbound blocked traffic.

Flexibility

Can be installed anywhere and given a domain name of your choice. Allows for greater security and customization.

Cost Friendly

Save costs for the service by self-hosting. License is still required, but plan is cheaper than cloud solution.

Mass Updates

Deploy software updates to a fleet of devices without necessarily needing internet connection.

Mass Device Analytics

Check the status of your devices without necessarily needing internet connection.

Mass Script Execution

Run scripts on a fleet of devices without necessarily needing internet connection.

Cloud Platform (Coming Soon)

Contains the web app to support offline use-cases, but also provides a suite of tools to effectively manage the devices that have internet. Part of our paid plan.

Offline Web App

Contains the web app to manage the IoT in the event that it goes offline for whatever reason.

Remote Terminal Access

Built-in compliance tools for GDPR, HIPAA, SOC 2, and other regulatory requirements with automated reporting and audit trails.

Remote File Transfer

Upload and download data from these devices using the remote file explorer.

Remote Updates

Mass update your fleet of devices by simply uploading a bundle artefact to the platform. Can all be done through a CI/CD pipeline.

Public URL

Allows a custom service running on the IoT to be accessible from over the internet with a public URL endpoint.

Device Alerts

Get notified when an IoT device is not behaving up-to-spec.

Frequently Asked Questions

Everything you need to know about our platform

How does the free plan differ from paid?

The free plan allows individually device management, while the paid plan allows fleet management.

Do I need special hardware?

No. Minimum system requirements are 2GB of ram, 8GB storage, and a 1 core CPU. Works on x86-64 or ARM64 systems.

How can I setup this service on my IoT device?

Please visit our setup guide for installation instructions, and visit our GitHub page for usage tips.

How can I use the self-hosted or cloud platform?

Right now the self-hosted and cloud platforms are still in development, so for the moment we only offer the offline option.

What operating systems are supported?

Our service provides full support for Linux-based OS, and partial support on other systems. Contact support for more details.

What makes you unique from other vendors?

Our primary differentiator is our support for network constraint IoT devices. For more details, please see our blog.
